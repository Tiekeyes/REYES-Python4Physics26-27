"The Remote Experience for Young Engineers and Scientists (REYES) virtual STEM-H learning experience aims to increase science literacy and inspire and train the next generation of engineers and scientists".
https://physics.berkeley.edu/visiting-students/reyes-remote-experience-young-engineers-and-scientists 

"The class, which begins on June 15 is designed to give students the key necessary tools to learn how to write simple code using a programming language called Python. You will learn to solve equations, do data analysis, and model various systems, among other techniques".
